Mobile Sale Dashboard

📱 Mobile Sales Analysis – Power BI Project

📌 Project Overview
This project focuses on analyzing mobile sales data using Power BI to uncover trends in revenue, sales quantity, brand performance, and regional distribution. The dashboard provides interactive insights that help in understanding customer demand patterns and business growth opportunities.

🎯 Business Problem Statement
The mobile retail business generates large volumes of transactional data, but raw data alone does not provide clear insights into sales performance, top-selling brands, or regional demand trends.
The business needs an interactive dashboard to:
Track overall sales and revenue performance
Identify top-performing mobile brands and models
Analyze regional and city-wise sales distribution
Monitor monthly and yearly growth trends
Support data-driven pricing and marketing decisions

🛠️ Tools & Technologies Used
Power BI
Data Cleaning & Transformation
DAX Functions
KPI Cards & Interactive Visualizations

📊 Key Insights
Identified top revenue-generating mobile brands
Compared sales performance across regions
Analyzed monthly growth trends
Evaluated quantity sold vs revenue contribution

🚀 Conclusion

This project demonstrates strong business intelligence and data visualization skills by transforming raw mobile sales data into meaningful insights that support strategic decision-making.
